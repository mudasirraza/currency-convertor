# README

### CURRENCY CONVERTOR

* Ruby version: 2.6.5

* Configuration:
Copy .env.example to .env and .env.test
RUN: bundle install

* Database creation/initialization
RUN: bundle exec rake db:setup

* How to run the test suite
RUN: bundle exec rspec

* Start development server
RUN: bundle exec rails s

* Job Que
RUN: bundle exec que
